About This Design System

MarineTrack is a fast, glanceable maritime navigation and dispatch UX designed for small vessel and fishing boat operators. The design system emphasizes safety, real-time telemetry, and one-tap actions with a delivery-first aesthetic inspired by ZUS Coffee's modern, warm, and inviting interface patterns.
