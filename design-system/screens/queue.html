Screen Purpose

Task management queue for dispatchers. Reorder, edit, and confirm multiple tasks before dispatch.

  • Header with total task count and clear queue option
  • Summary card showing total tasks, ETA window, affected vessels
  • Drag-to-reorder task cards (visual feedback during drag)
  • Each card: task type, vessel, priority, duration, edit/remove actions
  • Empty state when no tasks queued
  • Sticky Confirm Dispatch CTA at bottom
  • Success toast after dispatch confirmation
